What is DisplayRotate?

DisplayRotate is a powerful and lightweight jQuery plugin, which provides an easy way to rotate any content on your website. It also supplies some functions to manipulate it's usual behavior.

How can I use it?

Well, first you must ensure to include jQuery, the fast and concise JavaScript library, and the DisplayRotate.js file:

<script type="text/javascript" src="http://code.jquery.com/jquery-1.7.2.min.js"></script>
<script type="text/javascript" src="displayRotate.js"></script>

Next you need to initialize DisplayRotate. The easiest and quickest way to do this is as follows:

<script type="text/javascript">
  $(document).ready(function(){
    $("#your_display_container").displayRotate();
  });
</script>
